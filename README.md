# KUA Penjaringan Website

Proyek ini merupakan replika antarmuka dari situs KUA Penjaringan menggunakan **Vue 3**, **Vite**, dan **Tailwind CSS**.

## Prasyarat

- Node.js 18+
- npm 9+

## Pengembangan

```bash
npm install
npm run dev
```

Buka `http://localhost:5173` untuk melihat aplikasi.

## Build produksi

```bash
npm run build
npm run preview
```
